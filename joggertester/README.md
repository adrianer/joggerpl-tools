joggertester - narzędzie do testowania szablonów dla jogger.pl
--------------------------------------------------------------

Niniejsza aplikacja została stworzona w celu ułatwienia edycji szablonów dla
platformy blogowej jogger.pl w trybie offline. 

Aby jej użyć, skopiuj do katalogu "szablony" kod HTML Twojego szablonu. Plik 
z kodem strony głównej powinien mieć nazwę "glowna.html". Na dzień dzisiejszy 
konfiguracja skryptu polega na edycji pliku "views.py". Aby użyć joggertestera,
potrzebujesz zainstalowanego Pythona oraz bibliotekę Django.
